## Installation
Add this line to your application's Gemfile:

```ruby
gem 'spiral_form'
```

And then execute:
```bash
$ bundle install
```

Or install it yourself as:
```bash
$ gem install spiral_form
```

app/assets/stylesheets/application.scss

```scss
@import 'spiral_form';
 ```

app/assets/javascripts/application.js

```js
//=require spiral_form
 ```

view_file
```erb
<%= spiral_form %>
```
